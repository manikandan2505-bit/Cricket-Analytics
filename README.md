# Cricket Analytics Platform

A complete cricket analytics platform built using Python, Power BI, Machine Learning, and Streamlit.

## Features

- IPL and T20I player analytics
- Batting analysis dashboard
- Bowling analysis dashboard
- Player similarity engine
- Best XI generator
- Dream11 optimizer
- Anchor XI generation
- Attacking XI generation
- IPL vs T20I comparison
- Win probability prediction model
- Interactive Streamlit application
- Interactive Power BI dashboard

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- PuLP
- Streamlit
- Power BI

## Project Structure

```text
src/                 -> data processing scripts
exports/             -> generated analytics datasets
dashboard/           -> Power BI dashboard
app.py               -> Streamlit application
```

## Machine Learning

The project includes a win probability prediction model trained on IPL and T20I ball-by-ball data.

Model Accuracy:

```text
99.54%
```
